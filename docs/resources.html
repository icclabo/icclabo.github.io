Software

Convex Tools and Solvers:

  • CVX - A MATLAB-based modeling system used for defining and solving disciplined convex programs (DCPs).
  • SeDuMi - A Matlab/GNU Octave package for solving convex optimization problems involving linear equations and inequalities, second-order cone constraints, and semidefinite constraints (linear matrix inequalities).
  • SDPT3 - A MATLAB software for semidefinite-quadratic-linear programming.
  • Gurobi - A commercial mathematical optimization solver (engine) used for decision intelligence and prescriptive analytics.
  • MOSEK - A specialized, high-performance software package designed for solving large-scale mathematical optimization problems.
  • GLPK - A GNU Linear Programming Kit is a software package intended for solving large-scale linear programming, mixed integer programming, and other related problems.

Statistics Software:

  • wcc - An experimental R package for conducting windowed cross-correlation analyses.
  • varde - An experimental R package for decomposing the variance in mixed effects models.
  • agreement - An experimental R package for estimating inter-rater reliability.

Quarto Extensions

  • lordicon - A stable Quarto extension for embedding animated Lordicon icons in HTML.
  • honeypot - An experimental Quarto extension for detecting LLM cheating on HTML assignments.
  • details - A stable Quarto extension for embedding interactive details blocks in HTML documents.
  • embedpdf - A stable (but questioning) Quarto extension for embedding PDF files in HTML.

Docker Images:

  • rstudio2u - A multiarchitecture (amd64/arm64) image combining rstudio and r2u
  • rocker-bayes - A multiarchitecture (amd64/arm64) image for Bayesian analysis in R
  • wsl-whisper - A WSL (amd64) image for AI transcription from R with CUDA-support
